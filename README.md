## Projetos

Em meu perfil do GitHub há uma variedade de projetos que refletem meu interesse e paixão por resolver problemas por meio do código. Contém documentação da minha jornada de aprendizado, destacando os desafios enfrentados, lições aprendidas e pequenos testes e códigos exploratórios que desenvolvo para aprimorar minhas habilidades técnicas.

- [Meu Portfolio](https://github.com/Gerleidson/Projeto_MeuPortfolio): Aplicativo Android que permite aos usuários acessarem informações sobre o desenvolvedor, incluindo contato e links para suas redes sociais e projetos. Além disso, o aplicativo oferece acesso a seções específicas, como projetos, cursos e experiências.
- [Tributos VPB](https://github.com/Gerleidson/Projeto_TributosVPB): Aplicativo Android que permite calcular diferentes tipos de contribuições com base na remuneração e semeadura fornecidas pelo usuário. O aplicativo possui uma interface de usuário simples e intuitiva, e os resultados dos cálculos podem ser compartilhados com outras pessoas.
- [LTI Qualidade](https://github.com/Gerleidson/Projeto-LTI-Qualidade): Aplicativo Android que oferece acesso rápido e fácil a informações de contato e links para as redes sociais da empresa. Além disso, o aplicativo foi desenvolvido apenas para solicitação de serviços como manutenção, instalação e outros. Tudo isso sem cadastro.
